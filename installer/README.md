# Ralph Loop Installer

The guided wizard to install and manage the Ralph Loop—an agentic workflow orchestrator for Claude Code.

## Quick Start

```bash
npx @seevali/ralph-loop install
```

## What it does

- Scaffolds a Ralph Loop into an empty directory or existing project
- Installs BMAD (the Behavioral Model Driven development framework)
- Configures project conventions and environment
- Non-interactive mode for CI/CD scripts (`--yes`)

## Prerequisites

- Node.js 20 or later
- Git (strongly recommended; will warn if missing)
- Bash environment (Windows users: WSL2 required)
- jq (optional; some commands work better with it)
