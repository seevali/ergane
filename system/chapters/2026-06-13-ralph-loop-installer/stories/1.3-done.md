## Key Implementation Decisions

1. **Script follows the spec exactly.** The implementation matches the pseudocode in the story spec without deviation. No additional logic was added.

2. **`REPO_ROOT` computed from `BASH_SOURCE[0]`.** Two `..` levels up from `installer/scripts/` resolves to the repo root regardless of the caller's working directory — satisfying the edge case where the script is invoked from a subdirectory.

3. **`cmp -s` for byte comparison.** Silent, portable, and exits non-zero on any content difference. No dependency on `diff`, `md5sum`, or `sha256sum`.

4. **`find -print0` + `while IFS= read -r -d ''` loop.** Correctly handles filenames with spaces or special characters. Mirrors the source tree by stripping the `$SOURCE_PROMPTS` prefix and prepending `$DEST_PROMPTS`.

5. **Clean-slate prompts sync.** `rm -rf "$DEST_PROMPTS"` before `cp -rp` prevents stale files from accumulating across sync runs (e.g., if a prompt file is deleted from source).

6. **Variable quoting fix in check_mode.** The story spec's pseudocode had `"${src_file#$SOURCE_PROMPTS}"` without quoting `$SOURCE_PROMPTS`. The implementation uses `"${src_file#"$SOURCE_PROMPTS"}"` (inner quotes) — correct bash to avoid word-splitting on the pattern.
